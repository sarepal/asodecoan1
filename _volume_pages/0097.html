{% raw %}Junior Class Poem{% endraw %}
{% raw %}Only one short year has passed{% endraw %}
{% raw %}Since we were the Freshman (Mass,{% endraw %}
{% raw %}And how we'd splatter plaster was some fun.{% endraw %}
{% raw %}But now we 're really Juniors,{% endraw %}
{% raw %}One more year and we'll be Seniors,{% endraw %}
{% raw %}Then our funny splattering will be done.{% endraw %}
{% raw %}For we '11 have to brave the tide{% endraw %}
{% raw %}Without Hughes or Foster for our guide,{% endraw %}
{% raw %}And I fear sometimes the waves will beat us down;{% endraw %}
{% raw %}But with '' pluggers, " " probes'' and '' pullers''{% endraw %}
{% raw %}We can build a craft of '' dollars,''{% endraw %}
{% raw %}And if on this "boat," no one will ever drown.{% endraw %}
{% raw %}But just here 'tis sad to note{% endraw %}
{% raw %}That before we build our boat,{% endraw %}
{% raw %}We may have to ride one built by U. S. A.{% endraw %}
{% raw %}We won 't need our '' probes'' or " pullers,''{% endraw %}
{% raw %}Nor will we need any dollars,{% endraw %}
{% raw %}For on this boat our Uncle Sammy pays our way.{% endraw %}
{% raw %}And so if by luck or chance{% endraw %}
{% raw %}We safely land in France,{% endraw %}
{% raw %}We '11 still hope to be a D.D.S. some day.{% endraw %}
{% raw %}But if we shoulder guns{% endraw %}
{% raw %}And march out against the Huns{% endraw %}
{% raw %}All our Dental hopes will fade away.{% endraw %}
{% raw %}We '11 then think what kind of men{% endraw %}
{% raw %}We have, or should have been;{% endraw %}
{% raw %}And if our chance for a "better place" seems fair,{% endraw %}
{% raw %}Then our only hope or guide{% endraw %}
{% raw %}Will be in Him who rules the tide{% endraw %}
{% raw %}To bring us back or leave us '' over there.''{% endraw %}
{% raw %}Cecil A. Pless.{% endraw %}
