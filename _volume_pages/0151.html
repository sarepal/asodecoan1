{% raw %}Under no circumstances should the guest, when coffee in demi-tasses is served,{% endraw %}
{% raw %}ask the hostess for a large cup with hot milk. This is permissible at breakfast, but{% endraw %}
{% raw %}not at a formal dinner.{% endraw %}
{% raw %}GENERAL INFORMATION{% endraw %}
{% raw %}When cigarettes are furnished at the conclusion of a dinner, guests should use{% endraw %}
{% raw %}ash receivers for their butts and ashes, and not the table cloth or coffee cups.{% endraw %}
{% raw %}Neither should they strike matches on the furniture. The seat of the trousers is{% endraw %}
{% raw %}the proper place.{% endraw %}
{% raw %}Finger bowls are now taboo, for the reason that some diners seemed to often{% endraw %}
{% raw %}mistake them for drinking utensils, and others insisted on trying to take a bath{% endraw %}
{% raw %}in public.{% endraw %}
{% raw %}Tooth picks have also almost entirely been done away with, as it was found{% endraw %}
{% raw %}that diners abused this privilege. It is a hardship for the innocent to have to{% endraw %}
{% raw %}suffer with the guilty, but until the latter class can be taught to wield the tooth{% endraw %}
{% raw %}pick in a graceful manner, this useful article will have to be relegated to oblivion.{% endraw %}
{% raw %}If it be found absolutely necessary to use it, the operation should be performed in{% endraw %}
{% raw %}the cellar.{% endraw %}
{% raw %}Last, but not least, it should be said that wdien the liqueurs and cigarettes are{% endraw %}
{% raw %}served, guests should not so far forget themselves in their enjoyment as to put their{% endraw %}
{% raw %}feet on the table. If this be done, however, the host or hostess should be given the{% endraw %}
{% raw %}opportunity of removing the cloth and covering the table with pads to protect it{% endraw %}
{% raw %}from the heels of the diners.{% endraw %}
{% raw %}A writer of a letter to the editor wants to know "the proper way in which to{% endraw %}
{% raw %}use a knife and fork in eating a meal."{% endraw %}
{% raw %}The knife and fork should always be held in the hands, never with the toes.{% endraw %}
