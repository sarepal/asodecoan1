{% raw %}A Few Don'ts{% endraw %}
{% raw %}Don't say, '' I see right through you,''{% endraw %}
{% raw %}Although this may be true.{% endraw %}
{% raw %}While you are reading others,{% endraw %}
{% raw %}They, too, are reading you.{% endraw %}
{% raw %}Don't lean on other people,{% endraw %}
{% raw %}Put your shoulder to the wheel,{% endraw %}
{% raw %}You will soon find many missions{% endraw %}
{% raw %}At your shrine—to kneel.{% endraw %}
{% raw %}Don't forget to mind your business.{% endraw %}
{% raw %}Experience often shows{% endraw %}
{% raw %}We gain but small advantage{% endraw %}
{% raw %}When we tread on other 's toes.{% endraw %}
{% raw %}Don't neglect the laws of health,{% endraw %}
{% raw %}Whatever else you do—{% endraw %}
{% raw %}With this one priceless blessing lost{% endraw %}
{% raw %}Life holds but naught for you.{% endraw %}
{% raw %}Don't forget financial help{% endraw %}
{% raw %}When some friend needs a boost.{% endraw %}
{% raw %}Favors shown in need, like chickens{% endraw %}
{% raw %}Oft come home to roost.{% endraw %}
{% raw %}Don 't pour your woes and sorrows{% endraw %}
{% raw %}In other people's ears,{% endraw %}
{% raw %}They can't carry double burdens{% endraw %}
{% raw %}For their own increase with years.{% endraw %}
{% raw %}Don't allow repeated failures{% endraw %}
{% raw %}To discourage noble aim.{% endraw %}
{% raw %}Where there is a will a way is found{% endraw %}
{% raw %}Our object to attain.{% endraw %}
{% raw %}Marjorie Field.{% endraw %}
