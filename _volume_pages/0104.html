{% raw %}Freshman Class Poem{% endraw %}
{% raw %}A bright, industrious class{% endraw %}
{% raw %}Is the Class of '21,{% endraw %}
{% raw %}And the finals, we are sure to pass,{% endraw %}
{% raw %}For this is what we 've done.{% endraw %}
{% raw %}We have learned the use of phenol,{% endraw %}
{% raw %}And various other drugs,{% endraw %}
{% raw %}Which prevent the growth of all{% endraw %}
{% raw %}The microscopic '' bugs.''{% endraw %}
{% raw %}We've learned to carve a tooth,{% endraw %}
{% raw %}With all its grooves and ridges;{% endraw %}
{% raw %}To know that cleaning teeth in youth{% endraw %}
{% raw %}Prevents the use of bridges.{% endraw %}
{% raw %}In Prosthetics, each has done his bit;{% endraw %}
{% raw %}Our work now lies on a shelf.{% endraw %}
{% raw %}If you really don't believe it,{% endraw %}
{% raw %}Just go take a look yourself.{% endraw %}
{% raw %}Dentition we can give in full,{% endraw %}
{% raw %}And of the teeth describe their structure;{% endraw %}
{% raw %}And name the bones of the skull{% endraw %}
{% raw %}That form the lambdoid suture.{% endraw %}
{% raw %}We 've traced the process of cell division{% endraw %}
{% raw %}Through all its different phases;{% endraw %}
{% raw %}And we 've met with Dr. Hinman{% endraw %}
{% raw %}On his fifty-fifty basis.{% endraw %}
{% raw %}With all this learned, we will not '' bust,''{% endraw %}
{% raw %}But yet our work is mere' begun,{% endraw %}
{% raw %}For we are only Freshmenâ€”just{% endraw %}
{% raw %}The Class of '21.{% endraw %}
{% raw %}W. S. Duncan, Poet.{% endraw %}
