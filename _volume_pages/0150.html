{% raw %}ROASTS{% endraw %}
{% raw %}At all swagger affairs where roast goose is served, the guests are furnished with{% endraw %}
{% raw %}rubber coats, and in addition, the guests immediately to the right and left of the{% endraw %}
{% raw %}carver with armor plate, as the impact of a fat goose flying off a platfer while in{% endraw %}
{% raw %}the process of carving and coming into contact with a guest, is terriffie.{% endraw %}
{% raw %}This rule does not apply to Jewish affairs, however, as they have, by long{% endraw %}
{% raw %}experience, reduced the eating of a goose to a science.{% endraw %}
{% raw %}There are no special rules applicable to the eating of other roasts, except to{% endraw %}
{% raw %}sound the warning not to sop the bread in the gravy.{% endraw %}
{% raw %}The author will not make a separate chapter of vegetables, as they are usually{% endraw %}
{% raw %}served with the roast. The spoon should be used wherever possible, avoiding the{% endraw %}
{% raw %}use of the knife, except where peas are served. There are special square-cornered{% endraw %}
{% raw %}peas now being grown, as it was found very difficult to keep the ordinary pea from{% endraw %}
{% raw %}rolling off the knife. Where the old-fashioned garden variety is served, though, it{% endraw %}
{% raw %}will be found very convenient to mix them with the mashed potatoes. There is no{% endraw %}
{% raw %}more pitiful or harrowing sight than to see some famished diner securing about one{% endraw %}
{% raw %}pea in ten in the place they started for, and this can be avoided by the observance{% endraw %}
{% raw %}of the above rule.{% endraw %}
{% raw %}General, ordinary horse-sense should govern in the eating of other vegetables{% endraw %}
{% raw %}not specifically mentioned here. This, however, should not be interpreted as mean¬{% endraw %}
{% raw %}ing stable manners.{% endraw %}
{% raw %}SALADS{% endraw %}
{% raw %}Everything should be removed from the table, except the spoon, when the salad{% endraw %}
{% raw %}is served. Fingers are used exclusively now, and the spoon is resorted to only to{% endraw %}
{% raw %}get what dressing fails to stick to the salad when eaten. Formerly it was proper{% endraw %}
{% raw %}to eat lettuce with a fork, but people became too exasperated chasing a piece of{% endraw %}
{% raw %}lettuce from one side of the plate to the other, trying to get a foothold, so to speak,{% endraw %}
{% raw %}that its use has been entirely abandoned, and the better classes have gone back to{% endraw %}
{% raw %}Nature's own tools.{% endraw %}
{% raw %}DESSERTS{% endraw %}
{% raw %}The author regrets to say that he is not qualified to give extensive advice on{% endraw %}
{% raw %}this subject, as he, in common with many others, has largely given up the use of{% endraw %}
{% raw %}them for reasons that it is not necessary to go into detail. As far as he knows, they{% endraw %}
{% raw %}are still being eaten by less enlightened ones in the same old way. If this were a{% endraw %}
